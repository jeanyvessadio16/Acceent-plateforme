## Projet développé avec le Next js

## Dépendances installées

Tailwind CSS: pour le style
Shadcnui: pour utiliser des composants
Zustand: pour la validation des données
Lucide-icone: pour les icônes

## Pour l'exécution du l'application web

Saisir dans le terminal:
pnpm run dev

Ouvrir [http://localhost:3000](http://localhost:3000) dans le navigateur pour voir le resultat.

La page `app/page.tsx` est lancée.
